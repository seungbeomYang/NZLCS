# NZLCS Homepage Concepts - Frontend1

## Project
NZLCS 클라이언트 홈페이지 컨셉 시안 작업.
4개 컨셉 버전을 만들어서 클라이언트에게 전달하고, 방향 결정 후 본 작업 진행 예정.
현재는 시안 단계 — Sanity, Storybook 사용 안 함.


## Homepage
- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Brand
- Colors: Black, Burnt Orange, White
- Font: Josefin Sans
- public 폴더 안에 PDF Guideline.pdf를 통해 로고 이미지 색깔 확인 가능
- public안 Fiverr Premium Kit 폴더 안 PNG Logo Files 안에서 logo image 파일 확인가능

## Concepts (4 versions)
- Version1: Simple하게 갈 예정, reference 참고할만한 사이트로는 https://webflow.com/ 과 https://zoox.com/ 가 있음
- Primary color(#B25D1F)은 이미 확정된 상황에서 website의 seconday color는 #1A1A1A, Neutral(background)는 #F5F1EC, Accent는 #8B4513으로 사용
